CARACTERÍSTICAS DE LAS IMÁGENES
===============================

✓ Calidad profesional de galería
✓ Descripciones detalladas basadas en el catálogo PDF
✓ Efectos de iluminación profesional
✓ Variación de ángulos y perspectivas
✓ Detalles macro y de textura
✓ Optimizadas para web (JPG)

FUNCIONALIDADES IMPLEMENTADAS
==============================

1. Galería Interactiva
   - Click en miniaturas para seleccionar obra
   - Preview de imagen principal
   - Grid de 3 imágenes de detalle

2. Modal de Visualización
   - Click en imagen para ampliar
   - Navegación con flechas (anterior/siguiente)
   - Cierre con botón X o click fuera
   - Contador de imágenes
   - Transiciones suaves

3. Interactividad Visual
   - Hover effects en miniaturas
   - Escalado de imagen al pasar mouse
   - Indicadores visuales de zoom
   - Transiciones animadas

CÓMO FUNCIONA
=============

1. Usuario selecciona una obra en la grid principal
2. Se muestra la imagen principal + 3 detalles
3. User puede hacer click en cualquier imagen para ampliarla
4. En el modal:
   - Flechas izquierda/derecha para navegar
   - Número actual y total de imágenes
   - Click fuera del modal para cerrar
   - X button para cerrar

ARCHIVOS MODIFICADOS
====================

1. /lib/artist-data.ts
   - Interfaz Artwork actualizada con campos de imágenes
   - Todas las 8 esculturas tienen rutas de imágenes
   - mainImage: imagen principal (aspect-video)
   - detailImages: array de 3 imágenes de detalle

2. /components/gallery-section.tsx
   - Import de Image de Next.js
   - Actualización de interfaces para soportar imágenes
   - Gestión de estado para modal
   - Renderizado de imágenes con Image component
   - Modal fullscreen con navegación
   - Hover effects interactivos

PRÓXIMOS PASOS (OPCIONAL)
=========================

1. Personalizar con tus propias fotos:
   - Reemplazar archivos en /public/images/sculptures/
   - Mantener los mismos nombres de archivo
   - Preferible JPG o WebP

2. Agregar más imágenes:
   - Actualizar array detailImages en artist-data.ts
   - Agregar archivos en /public/images/sculptures/

3. Agregar imágenes de pinturas:
   - Similar a esculturas
   - Crear /public/images/paintings/
   - Actualizar paintingWorks en artist-data.ts

NOTAS IMPORTANTES
