🚀 CÓMO EMPEZAR (3 opciones)
═════════════════════════════════════════════════════════════════════════════════

⚡ OPCIÓN 1: Rápido (5 minutos)
   1. npm install
   2. npm run dev
   3. Ve a http://localhost:3000
   📖 Lee: QUICK_START.md

📖 OPCIÓN 2: Completo (10-15 minutos)
   1. npm install
   2. npm run dev
   3. Edita /lib/artist-data.ts
   4. Agrega imágenes a /public/images/
   📖 Lee: README_PROYECTO.md

🚀 OPCIÓN 3: Desplegar (30 minutos)
   1. Completa opciones 1 o 2
   2. Crea repo en GitHub
   3. Ve a vercel.com/new
   4. Conecta repo
   5. Haz clic Deploy
   📖 Lee: DEPLOYMENT.md
