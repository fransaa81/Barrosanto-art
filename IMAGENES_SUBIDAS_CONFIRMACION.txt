🚀 CÓMO VER LAS IMÁGENES:

1. Abre terminal y ejecuta:
   npm run dev

2. Abre http://localhost:3000 en tu navegador

3. Navega a la sección "Obra Escultórica"

4. Haz click en cualquier obra para ver la galería

5. Haz click en una imagen para ampliarla

6. Usa flechas para navegar o X para cerrar
