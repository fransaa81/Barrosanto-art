## 🎯 Rutas de Aprendizaje

### Ruta 1: "Quiero Empezar Ahora" (30 min)
1. Leer: `QUICK_START.md`
2. Ejecutar: `npm install && npm run dev`
3. Cambiar contenido en `/lib/artist-data.ts`
4. Agregar fotos en `/public/images/`

### Ruta 2: "Quiero Entenderlo Todo" (1 hora)
1. Leer: `README_PROYECTO.md`
2. Leer: `PORTFOLIO_README.md`
3. Explorar estructura de carpetas
4. Revisar `/lib/artist-data.ts`
5. Revisar componentes en `/components/`

### Ruta 3: "Quiero Personalizar Mucho" (1-2 horas)
1. Completar Ruta 1
2. Leer: `IMAGES_SETUP.md`
3. Leer: `EXTENSION_EXAMPLES.md`
4. Agregar imágenes
5. Agregar nuevas características

### Ruta 4: "Quiero Desplegar" (30 min)
1. Completar Ruta 1
2. Leer: `DEPLOYMENT.md` - Opción 1
3. Crear repo en GitHub
4. Desplegar en Vercel
5. Configurar dominio personalizado
